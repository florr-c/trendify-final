Trendify - E-commerce:
Trendify es una tienda en línea de artículos de tecnología, desarrollada con React y utilizando Firebase para el almacenamiento de productos y la gestión de autenticación de usuarios. La aplicación funciona en un entorno de React con React Router para la navegación.

* Requisitos de instalación:
Para ejecutar el proyecto localmente,hay que instalar:

- Node.js, para poder ejecutar React
- npm o yarn para la gestión de paquetes
- Firebase CLI para la configuración del backend
